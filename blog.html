Discuss the scope of var, let, and const

var declarations are global/functional scoped based on context. The scope is global when a var variable is declared outside a function. var is function scoped when it is declared within a function. Variables declared with var can be reassigned.

let declarations are block scoped. So a variable declared in a block with let is only available for use within that block. Variables declared with let can be reassigned.

const declarations are block scoped. So a variable declared in a block with const is only available for use within that block. Variables declared with const cannot be reassigned.

Tell us the use cases of null and undefined

null is trivial/other type of variable. null should be used when we want to explicitly set a value to a variable which does not exist. The type of null is object.

undefined is a variable that has been declared but not initialized. We can get undefined through several ways such as: with a function with no return, accessing array elements outside of index range etc.

What do you mean by REST API?

A REST API is a way of easily accessing web services which allows interaction with resources such as fetching information from a server over the internet to build web application, mobile application and other services.
